The jmx-over-ssl template includes the following changes from the default tc
Runtime instance:
  - A modified server.xml that uses SSL for all JMX communication.
  - A sample JKS keystore that can be used to test the JMX over SSL
    configuration. The sample JKS keystore is not suitable for production
    systems.
